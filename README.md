## Getting Started

```bash
$ git clone https://github.com/Zerokei/AskGPT.git
```

## Usage

```bash
$ python3 src/main.py -R  # create config.yaml
$ python3 src/main.py -A  # add an api-key
$ python3 src/main.py -S  # select api-key
$ python3 src/main.py # ask GPT
```
